3.6.8.3 Forced Output Alignment

You can increase an output section's alignment by using ALIGN. As an alternative you can force the output section alignment to the maximum alignment of all its input sections with ALIGN_WITH_INPUT. The alignment forced by ALIGN_WITH_INPUT is used even in case the load and virtual memory regions are different.
